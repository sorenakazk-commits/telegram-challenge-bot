# ربات چالش سرعتی تلگرام (UserBot)

رباتی که روی اکانت شخصی شما فعال می‌شود و در چالش‌های سرعتی کانال، کلمه را سریع‌تر از همه به پیوی شخص چالش‌دهنده می‌فرستد.

## نحوه کار

ربات دو حالت دارد که با فرمان در **Saved Messages** (پیام‌های ذخیره‌شده) کنترل می‌شود:

| فرمان | حالت | عملکرد |
|-------|------|--------|
| `آماده` | الگوی A | ربات مسلح می‌شود؛ به محض رسیدن پیام بعدی کانال که «کلمه» باشد، فوراً آن را به پیوی شخص می‌فرستد |
| `کلمه R8A 20:45` | الگوی B | کلمه `R8A` را رأس ساعت `20:45:00` (به وقت ایران) به پیوی شخص می‌فرستد |
| `لغو` | — | هر دو حالت مسلح/زمان‌بندی‌شده را لغو می‌کند |
| `وضعیت` | — | وضعیت فعلی ربات را نمایش می‌دهد |

> توجه: در الگوی A ربات پیام‌های اعلان/اطلاعاتی (حاوی کلماتی مثل چالش، ساعت، جایزه، عضو، فوروارد و...) را نادیده می‌گیرد و فقط پیام «کلمه» را ارسال می‌کند. ارسال به صورت **متن خام** است (نه فوروارد).

## پیش‌نیازها

1. پایتون ۳.۱۰ به بالا (روی سیستم خودتان برای ساخت session)
2. یک اکانت Telegram
3. حساب Railway (railway.app)

## گام اول: گرفتن API_ID و API_HASH

1. به آدرس https://my.telegram.org بروید
2. وارد شوید با شماره تلفن تلگرام‌تان
3. بخش **API development tools** → یک اپلیکیشن جدید بسازید
4. `api_id` و `api_hash` را یادداشت کنید

## گام دوم: ساخت SESSION_STRING (روی سیستم خودتان)

```bash
pip install telethon
python generate_session.py
```

- شماره تلفن (با کد کشور، مثلاً `989123456789`) و کد تایید را وارد کنید
- یک رشته طولانی چاپ می‌شود → آن را کپی کنید (این `SESSION_STRING` است)
- ⚠️ این رشته را محرمانه نگه دارید؛ دسترسی کامل به اکانت شما را می‌دهد

## گام سوم: آیدی‌های مورد نیاز

- **CHAT_ID**: آیدی عددی کانال چالش
  - راحت‌ترین راه: پیام ربات [@userinfobot](https://t.me/userinfobot) را فوروارد کنید یا از کلاینت‌هایی که آیدی عددی نشان می‌دهند استفاده کنید
  - فرمت کانال: `-1001234567890` (با پیشوند `-100`)
- **TARGET_PM_ID**: آیدی عددی شخص چالش‌دهنده (R8A)
  - از [@userinfobot](https://t.me/userinfobot) بگیرید یا به پیوی او پیام بدهید و آیدی را استخراج کنید

## گام چهارم: دیپلوی روی Railway

1. وارد railway.app شوید → **New Project** → **Deploy from GitHub repo** (یا Deploy from repo)
2. فایل‌های این پروژه را در ریپوی خود push کنید
3. در تب **Variables** متغیرهای زیر را اضافه کنید:

   | متغیر | توضیح |
   |-------|-------|
   | `API_ID` | api_id از my.telegram.org |
   | `API_HASH` | api_hash از my.telegram.org |
   | `SESSION_STRING` | رشته‌ای که در گام دوم گرفتید |
   | `CHAT_ID` | آیدی عددی کانال (مثلاً `-1001234567890`) |
   | `TARGET_PM_ID` | آیدی عددی شخص چالش‌دهنده |
   | `CONTROL_CHAT` | (اختیاری) پیش‌فرض `me` |

4. Railway به‌صورت خودکار `requirements.txt` را نصب و `python main.py` را اجرا می‌کند
5. در تب **Logs** باید ببینید: `ربات آنلاین شد`

## گام پنجم: استفاده

1. در تلگرام به **Saved Messages** بروید
2. برای الگوی A: حدود ۲ دقیقه قبل از چالش بنویسید `آماده`
3. برای الگوی B: بنویسید `کلمه R8A 20:45` (کلمه و ساعت دلخواه)
4. ربات خودکار کلمه را به پیوی شخص می‌فرستد

## نکات سرعت

- کلاینت Telethon همیشه متصل می‌ماند (بدون delay اتصال مجدد در لحظه حساس)
- در الگوی A ارسال در همان event handler و بدون مکث انجام می‌شود
- در الگوی B تا ۰.۵ ثانیه قبل با `sleep` و سپس با busy-wait دقیق تا ثانیه مقرر صبر می‌کند
- زمان‌بندی بر اساس `Asia/Tehran` است

## هشدار

- استفاده از UserBot ممکن است با قوانین تلگرام در تضاد باشد؛ مسئولیت با شماست
- SESSION_STRING را هرگز در جای عمومی (مثل GitHub) قرار ندهید
